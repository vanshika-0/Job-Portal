# Job Portal

## Description
A web-based platform that allows job seekers to register, search, and apply for jobs online.

## Features
- User registration and login
- Job search and filters
- Upload resume
- Employer job posting

## Tech Stack
- Spring Boot
- MySQL
- Maven
- HTML, CSS, JS

## Installation
1. Clone the repository.
2. Import project into IDE.
3. Configure database.
4. Run using embedded Apache Tomcat server.
